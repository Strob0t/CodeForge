## Status Legend

- PASS -- tests exist and pass
- FAIL -- tests exist but failing
- NONE -- no tests exist for this layer
- N/A -- not applicable for this layer
- SKIP -- intentionally skipped (documented reason)

## Verification Criteria

A feature is **verified** when:
1. All applicable test layers show PASS
2. Contract tests pass for any NATS payload the feature uses
3. At least one smoke flow exercises the feature end-to-end

A feature is **partially verified** when:
1. Some but not all applicable test layers show PASS
2. No FAIL results in any layer

## Test File Reference

| Test Type | Location | Run Command |
|-----------|----------|-------------|
| Go unit tests | `internal/**/*_test.go` | `go test ./internal/...` |
| Go integration | `tests/integration/*_test.go` | `go test -tags=integration ./tests/integration/...` |
| Go smoke tests | `tests/integration/smoke_test.go` | `go test -tags=smoke -timeout=300s ./tests/integration/...` |
| Go contract tests | `internal/port/messagequeue/contract_test.go` | `go test ./internal/port/messagequeue/ -run Contract` |
| Python unit tests | `workers/tests/test_*.py` | `cd workers && poetry run pytest` |
| Python contract tests | `workers/tests/test_nats_contracts.py` | `cd workers && poetry run pytest tests/test_nats_contracts.py` |
| Frontend E2E (browser) | `frontend/e2e/*.spec.ts` | `cd frontend && npx playwright test` |
| Frontend E2E (API) | `frontend/e2e/api/*.spec.ts` | `cd frontend && npx playwright test e2e/api/` |
| Frontend E2E (LLM) | `frontend/e2e/llm/*.spec.ts` | `cd frontend && npx playwright test --config=playwright.llm.config.ts` |
| Frontend E2E (NATS) | `frontend/e2e/nats/*.spec.ts` | `cd frontend && npx playwright test e2e/nats/` |
